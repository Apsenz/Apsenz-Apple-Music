# Support

## Recommended version

Use the Vencord plugin if you already use Vencord.

## Install Vencord Plugin

1. Download `ApsenzAppleMusic_Vencord_Plugin.zip` from the latest release.
2. Extract the ZIP.
3. Run `ApsenzAppleMusic-installer.bat`.
4. Restart Discord.
5. Enable `ApsenzAppleMusic` in Vencord plugins.

## Uninstall Vencord Plugin

Run `ApsenzAppleMusic-uninstaller.bat`.

## Windows Desktop App

Download `Apsenz-Apple-Music-windows-x64-installer.exe` from the latest release and install it normally.

## Troubleshooting

If the Vencord plugin does not appear:

1. Fully close Discord.
2. Run the installer again.
3. Make sure Vencord is installed.
4. Restart Discord.

If the installer fails, check the log file:

`%TEMP%\ApsenzAppleMusic-installer.log`
