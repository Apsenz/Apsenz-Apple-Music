# Security Policy

## Supported Versions

Only the latest release is supported.

Download the latest version here:

https://github.com/Apsenz/Apsenz-Apple-Music/releases/latest

## Reporting a Security Issue

If you find a security issue, do not open a public issue with sensitive details.

Contact through the Discord server:

https://discord.gg/k5zFTFjEhU

## Privacy Scope

Apsenz Apple Music is designed to read Apple Music playback information only.

It should not use browser media sessions, YouTube, video players, or other apps for Discord Rich Presence.
