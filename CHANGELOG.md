# Changelog

## v1.0.0

Initial public release.

### Added

- Vencord plugin for Apple Music Rich Presence.
- Windows desktop app installer.
- Discord Rich Presence with song title, artist, album, artwork, and progress.
- Self-contained Vencord plugin installer and uninstaller.
- Preview screenshots.

### Privacy fix

- Only Apple Music is used for Rich Presence.
- Browser media, YouTube, videos, and other media sessions are ignored.
